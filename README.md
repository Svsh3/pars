# 🛍️ WB Parser Bot

Telegram-бот для парсинга цен с Wildberries. Принимает `.txt` файл с запросами → возвращает Excel с ценами в формате сводной таблицы.

---

## ⚡ Быстрый старт

### 1. Получи токен бота

1. Напиши [@BotFather](https://t.me/BotFather) в Telegram
2. `/newbot` → придумай имя и username
3. Скопируй токен вида `7654321098:AAH...`

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

### 3. Вставь токен в bot.py

Открой `bot.py` и замени строку:
```python
BOT_TOKEN = os.getenv("BOT_TOKEN", "ВСТАВЬ_СВОЙ_ТОКЕН_СЮДА")
```
либо:
- **Вариант A** — вставь токен напрямую: `BOT_TOKEN = "7654321098:AAH..."`
- **Вариант B** — задай переменную окружения: `export BOT_TOKEN=7654321098:AAH...`

### 4. Запусти бота

```bash
python bot.py
```

---

## 🤖 Как использовать бота

1. Найди своего бота в Telegram
2. Напиши `/start`
3. Введи пароль: **`qw1`**
4. Создай файл `queries.txt`:
   ```
   беспроводные наушники
   механическая клавиатура
   умные часы
   ```
5. Отправь файл боту
6. Получи Excel 🎉

## 🚀 Деплой на Railway / VPS

### Railway (бесплатно)
1. Залей папку на GitHub
2. Зайди на [railway.app](https://railway.app)
3. New Project → Deploy from GitHub
4. Добавь переменную: `BOT_TOKEN = твой_токен`
5. Готово — бот работает 24/7

### Добавь `Procfile` для Railway:
```
worker: python bot.py
```
